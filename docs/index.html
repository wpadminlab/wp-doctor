wp-doctor

Auditez la sécurité, la performance et le SEO de n'importe quel site WordPress — en une seule commande, à distance, sans rien installer.

⭐ Voir sur GitHub Voir un rapport d'exemple

Démarrage en 10 secondes

# Sans installation, avec Python 3.8+
python3 -m wpdoctor exemple.com

# Ou installez la commande
pip install wp-doctor
wp-doctor exemple.com --format html -o rapport.html

Zéro dépendance : uniquement la bibliothèque standard de Python.

Trois audits en un

🔒

Sécurité

Fichiers sensibles exposés, xmlrpc ouvert, énumération d'utilisateurs, en-têtes de sécurité manquants, version exposée.

⚡

Performance

Temps de réponse serveur, cache de page, compression, poids du HTML, lazy-loading des images.

🔍

SEO technique

Title, meta description, canonical, Open Graph, données structurées, robots.txt, sitemap, noindex accidentel.

Ce que wp-doctor contrôle

  • Fichiers .env, wp-config.php.bak, .git/config
  • xmlrpc.php et surface de brute-force
  • Énumération via API REST et ?author=1
  • HSTS, CSP, X-Frame-Options, nosniff
  • Version WordPress et empreinte des plugins
  • TTFB et cache de page (LiteSpeed, CDN…)
  • Compression gzip / brotli
  • Données structurées JSON-LD
  • Sitemap et robots.txt cohérents
  • Score /100 et note A→F exportable
Usage responsable. wp-doctor est passif et non intrusif (aucune exploitation, aucun brute-force). N'auditez que des sites qui vous appartiennent ou pour lesquels vous avez une autorisation explicite.

Rapport clair, exportable

Sortie terminal colorée, ou export HTML, Markdown et JSON pour l'intégrer à vos dashboards ou votre CI (--fail-on critical).
